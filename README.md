# ⚡ BESCOM EV Charging Load Forecast Dashboard

An interactive dashboard to forecast EV charging load 
on BESCOM's electricity grid from 2026 to 2030.

## 🔗 Live Dashboard
[Click here to open the dashboard](https://bescom-ev-dashboard.streamlit.app)

## 📊 What it does
- Forecasts hourly EV charging load using real BESCOM data
- Projects grid demand from 2026 to 2030
- Built with Polynomial Linear Regression (R² = 0.634)

## 🛠️ Built with
- Python, Streamlit, Matplotlib, Scikit-learn
- Real data from BESCOM EV Cell (May 2026)
